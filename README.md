# Project Resonance

> **The software should disappear. The music should remain.**

Project Resonance is an open-source iPhone companion for the NUX Mighty Plug Pro MP-3.

Its purpose is simple:

**Help guitarists spend less time configuring equipment and more time creating music.**

Unlike traditional amplifier control applications, Project Resonance is designed around the creative workflow of musicians rather than the technical capabilities of hardware.

---

## Mission

Help guitarists capture inspiration, preserve great tones, and recreate them whenever inspiration strikes again.

---

## Guiding Principle

> **Every decision should make it easier to create music.**
